## الخطوة 1: تشغيل المترجم

### تشغيل ملف اختبار بسيط:

```bash
cd interpreter/
python3 aether_interpreter.py test.ae
```

**الناتج:**
```
أهلاً بالعالم
مجموع 5 + 10 هو:

```

### تشغيل ملف اختبار متقدم:

```bash
python3 aether_interpreter.py advanced_test.ae
```

**الناتج:**

```
مرحباً بك في لغة نيكسو

مساحة الدائرة بنصف قطر 5 هي:
78.53981633974483
اسم اللغة بالأحرف الكبيرة: نيكسو
```

## الخطوة 2: كتابة برنامج أول بك

### إنشاء ملف جديد `hello.nexo`:

```nexo
let greeting = "مرحباً بك في نيكسو!"
print(greeting)

let x = 10
let y = 20
print("مجموع " + str(x) + " و " + str(y) + " هو: " + str(x + y))
```

### تشغيل البرنامج:

```bash
python3 aether_interpreter.py hello.nexo
```

## الخطوة 3: تشغيل الموقع الإلكتروني

### المتطلبات:
- Node.js 22.x أو أحدث
- pnpm

### التثبيت والتشغيل:

```bash
cd website/nexo_lang_website/
pnpm install
pnpm dev
```

### الوصول للموقع:
افتح المتصفح على: **http://localhost:3000**

### المميزات المتاحة:
- ✅ الصفحة الرئيسية مع عرض المميزات

## أمثلة سريعة

### مثال 1: الرياضيات

```nexo
let pi = math_pi()
let radius = 5
let area = pi * math_pow(radius, 2)
print("مساحة الدائرة: " + str(area))
```

### مثال 2: النصوص

```nexo
let name = "نيكسو"
let upper = str_upper(name)
let lower = str_lower(name)
print("الأصلي: " + name)
print("كبير: " + upper)
print("صغير: " + lower)
```

### مثال 3: الحلقات

```nexo
for i in 0..5 {
    print("العد: " + str(i))
}
```

### مثال 4: الشروط

```nexo
let age = 25
if age >= 18 {
    print("بالغ")
} else {
    print("قاصر")
}
```

## استكشاف الأخطاء

### المشكلة: "Python not found"
**الحل:** تأكد من تثبيت Python 3.x على نظامك
```bash
python3 --version
```

### المشكلة: "File not found"
**الحل:** تأكد من أنك في المجلد الصحيح
```bash
cd interpreter/
ls *.ae
```

### المشكلة: خطأ في الموقع الإلكتروني
**الحل:** تأكد من تثبيت التبعيات
```bash
cd website/nexo_lang_website/
pnpm install
```
