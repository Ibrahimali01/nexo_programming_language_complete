# 🚀 لغة نيكسو (Nexo) - النسخة الكاملة والمستقلة

## ✨ ما هي نيكسو؟

**نيكسو** هي لغة برمجة **ثورية ومستقلة تماماً** تجمع بين أفضل مميزات لغات البرمجة العالمية:

- ✅ **بساطة Python** - سهولة القراءة والكتابة
- ✅ **قوة Rust** - الأمان والأداء العالي
- ✅ **مرونة JavaScript** - التطوير السريع
- ✅ **شمولية Java** - دعم جميع المنصات

## 💻 أمثلة الكود

### مثال 1: الدوال

```nexo
func add(a: Int, b: Int) -> Int {
    return a + b
}

let result = add(10, 20)
print(result)  // 30
```

### مثال 2: الشروط والحلقات

```nexo
for i in 1..10 {
    if i % 2 == 0 {
        print(i)
    }
}
```

### مثال 3: البرمجة الكائنية

```nexo
class Person {
    let name: String
    
    constructor(name: String) {
        this.name = name
    }
    
    func greet() -> String {
        return "مرحباً " + this.name
    }
}

let person = new Person("أحمد")
print(person.greet())
```

### مثال 4: معالجة الملفات

```nexo
import fs

let content = fs.read("file.txt")
fs.write("output.txt", content)
```

### مثال 5: قاعدة بيانات

```nexo
import database

let db = database.connect("sqlite:///mydb.db")
db.insert("users", {"name": "أحمد", "age": 30})
db.close()
```
